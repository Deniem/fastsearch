Fast search

Fast search is lightweight and responsive quick-search and auto-suggest library based on jQuery api. Build smart search UI controls that can easily connect to back-end api and enhance user experience. It weighs about 2.5KB (minified and gziped).

Installation

Browse dist folder for library JS and CSS files. Fastsearch js file (normal or minified one) should be included after jQuery flavor of your choice if you go with plain script tags. You can use provided or build your own css.

Download library files from github repo or get them via bower (bower install fastsearch)
